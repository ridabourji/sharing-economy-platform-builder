== Installation ==
NOTE: WCMp For Job Manager plugin is an extension of WC Marketplace,  so the WC Marketplace plugin must be installed and activated in your WordPress site for this plugin to work properly.

1. Download and install WCMp Job Manager using the built-in WordPress plugin installer.
If you download WCMp Job Manager plugin manually, make sure it is uploaded to `/wp-content/plugins/` and activate the plugin through the `Plugins` menu in WordPress. Or follow the steps below:
Plugins > Add new > Upload plugin > Upload wcmp-job-manager.zip > Install Now > Activate Plugin.
2. To activate Job Listing
Go to WooCommerce >> WCMp >> Capabilities >> WPMp Job Manager
3. To activate Job Package
Go to WooCommerce >> WCMp >> Products >> Product Types
